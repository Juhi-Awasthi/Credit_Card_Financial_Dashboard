# 📊 Credit Card Financial Dashboard

## 📌 Overview
The Credit Card Financial Dashboard is a business intelligence project designed to analyze credit card transactions, customer behavior, and financial performance.

This project leverages SQL and Power BI to transform raw transactional data into meaningful insights, enabling data-driven decision-making.

---

## 🎯 Business Objective
- Monitor overall revenue, transactions, and interest earnings  
- Analyze customer demographics and spending behavior  
- Track quarterly and weekly performance trends  
- Identify high-value customer segments  

---

## 🛠️ Tech Stack
- SQL (MySQL / PostgreSQL) – Data querying and transformation  
- Power BI – Dashboard visualization  
- DAX – KPI calculations  
- CSV / Excel – Data sources  

---

## 🔄 Workflow
1. Data collection from CSV files  
2. Data cleaning and transformation using SQL  
3. Data modeling in Power BI  
4. KPI creation using DAX  
5. Dashboard development  
6. Insight generation  

---

## 📊 Dashboard Insights

### 💳 Transaction Analysis
- Total Revenue: 55.4M  
- Total Transactions: 657K  
- Transaction Amount: 45M  
- Total Interest Earned: 7.9M  

#### Key Observations:
- Swipe transactions generate the highest revenue  
- Blue card category contributes the most revenue  
- Top spending categories include Bills, Entertainment, Fuel, and Grocery  
- Quarterly performance remains stable with peak in Q3  

---

### 👥 Customer Analysis
- Total Revenue: 55.4M  
- Total Income: 577M  
- Customer Satisfaction Score: 3.19  

#### Key Observations:
- Age group 30–40 contributes the highest revenue  
- Female customers generate slightly higher revenue than males  
- High-income customers drive the majority of revenue  
- Self-employed and businessmen are top contributors  

---

## 📈 Key Insights
- Revenue is highly concentrated among Blue card users  
- Middle-aged and high-income customers are most valuable  
- Offline transactions dominate digital transactions  
- Customer acquisition cost is highest for Blue category  
- Stable quarterly growth indicates consistent performance
